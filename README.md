## 📖 O sadě

Tento repozitář obsahuje komplexní data ve formátu **OQSE** (Open Question Set Exchange). 

Sada postupně mapuje celé lidské tělo. Místo biflování z učebnice si zde můžete struktury prohlédnout ze všech úhlů.
* **Typy otázek:** `pin-on-model` (Uživatel musí identifikovat a kliknout na správný `Mesh` v `.glb` souboru).
* **Pokrytá témata:** Osteologie, myologie, orgánové soustavy a inervace.
* **Assety:** Součástí repozitáře je složka `/assets` obsahující optimalizované GLTF/GLB modely jednotlivých částí těla (končetiny, lebka, trup atd.).

## 🚀 Jak sadu používat

Sada je plně kompatibilní s ekosystémem Memizy. Pro spuštění nemusíte stahovat gigabyty dat. Stačí si v aplikaci Memizy otevřít import a vložit přímý odkaz na data této sady:
[https://cdn.jsdelivr.net/gh/memizy/set-anatomie-3d@main/data.oqse.json](https://cdn.jsdelivr.net/gh/memizy/set-anatomie-3d@main/data.oqse.json)
